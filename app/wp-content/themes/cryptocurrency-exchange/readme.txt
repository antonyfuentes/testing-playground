Cryptocurrency Exchange WordPress theme is a fully responsive theme that looks great on any devices including iPad, iPhone or any other mobile device.

== Installation ==
Manual installation:

1. Upload the `cryptocurrency-exchange` folder to the `/wp-content/themes/` directory

Activiation and Use

1. Activate the Theme through the 'Themes' menu in WordPress
2. See Customizer-> cryptocurrency-exchange Theme Options to change theme specific options

== Custom Logo ==
This theme supports custom logo for the header part. 
You can change or remove it from Customizer -> Theme Options -> Header Settings.
